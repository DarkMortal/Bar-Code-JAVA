# <ins>This project allows us to create Barcodes of various formats with ease</ins>

## _**Codes supported by the app**_ 

* ### Code 39
* ### Code 128
* ###  UPC-A
* ### EAN8
* ### EAN13
* ### PDF417
* ### QR Code
